# Xbanner
生成旗帜的小插件，导入便可用点击
将Xbanner.js存放在您的项目中<br>

```HTML结构
      <div id =“test1”>
        <div>
            <img src="./img/banner1.jpg" alt="">
            <img src="./img/banner2.jpg" alt="">
            <img src="./img/banner3.jpg" alt="">
            <img src="./img/banner4.jpg" alt="">
        </div>
    </div>
```

```JS创造实
    新的Xbanner（ID，宽度，高度)
    var a  = new Xbanner('test1',600,300)
```
